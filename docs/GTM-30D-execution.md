# TalkToBro — 30 Day GTM Execution Plan

## North Star
- Drive consistent trial starts and convert fast to paid.
- Build distribution moat on X + outbound email.

## Weekly Targets
- 70+ new trials/week
- 10+ new paid/week
- 10 booked calls/week from outbound

## Channel 1: X Growth Engine (Daily)
1. 1 flagship post/day (build in public, proof, hard opinion)
2. 8-12 reply-guy replies/day on high-reach posts in AI/ops/SMB automation
3. 1 proof artifact every 48h (screenshot/result/workflow)
4. 1 CTA post/day (trial link)

## Channel 2: Maps -> Trello -> Email Engine (Daily)
1. Pull 40-100 new local leads/day from Google Places
2. Push into Trello list: New Leads
3. Run outreach in controlled batches (20/day to start)
4. Move cards by state: Contacted -> Responded -> Demo Scheduled -> Closed Won

## Offer Narrative
- Start free for 24 hours.
- Keep Bro for £15/month if valuable.
- Positioning: "Bro that actually does things".

## KPI Dashboard
- Trials (24h): count
- New paid (24h): count
- Trials -> Paid conversion (7d)
- Emails sent/day
- Reply rate
- Demos booked/week

## Non-negotiables
- Clear CTA language: free trial first, payment second.
- Personal touch for every new payer within 15 minutes.
- Track everything in one place.
